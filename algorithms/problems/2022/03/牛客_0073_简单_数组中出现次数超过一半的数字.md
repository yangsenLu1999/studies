<summary><b>思路：“摩尔投票”</b></summary>

<details><summary><b>Python</b></summary>

```python
class Solution:
    def MoreThanHalfNum_Solution(self , arr: List[int]) -> int:
        
        ret = arr[0]
        cnt = 1
        
        for x in arr[1:]:
            if x == ret:
                cnt += 1
            else:
                cnt -= 1
            
            if cnt == 0:
                ret = x
                cnt = 1
        
        return ret
```
